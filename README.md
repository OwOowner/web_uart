# Serial Monitor Pro - 串口调试工具

一个功能强大的现代化串口调试工具，基于Web Serial API开发，提供实时监控、数据分析和配置管理等功能。

## 🌟 主要功能

### 实时监控
- **自动串口检测**: 自动扫描并识别所有连接的串口设备
- **实时数据显示**: 支持文本、十六进制、二进制等多种显示格式
- **多串口监控**: 同时监控多个串口的通信状态
- **数据过滤**: 按串口、数据类型、关键字等条件过滤显示

### 配置管理
- **详细参数配置**: 支持波特率、数据位、停止位、校验位等完整配置
- **预设管理**: 保存和加载常用配置方案
- **连接测试**: 发送测试数据验证串口配置
- **批量操作**: 同时对多个串口进行配置和操作

### 数据分析
- **实时图表**: 数据流量、错误率等统计可视化
- **协议分析**: 自动识别和分析常见通信协议
- **日志管理**: 实时日志显示、过滤和导出
- **报告生成**: 生成详细的通信分析报告

### 数据导出
- **多种格式**: 支持CSV、JSON、文本、十六进制等格式导出
- **时间范围**: 按时间范围筛选导出数据
- **报告生成**: 生成汇总报告、详细报告、错误分析等

## 🚀 技术特性

### 现代技术栈
- **Web Serial API**: 基于现代浏览器原生串口通信API
- **响应式设计**: 适配桌面、平板、移动设备

## 🎯 使用指南

### 环境要求
- **浏览器**: Chrome 89+ 或 Edge 89+
- **操作系统**: Windows, macOS, Linux
- **硬件**: 需要连接串口设备

### 快速开始
1. **启动工具**: 在支持的浏览器中打开 `index.html`
2. **扫描串口**: 点击"扫描串口"按钮检测可用设备
3. **连接设备**: 选择串口并点击"连接"
4. **开始监控**: 实时查看串口通信数据

### 高级功能
1. **配置串口**: 访问配置页面设置详细参数
2. **数据分析**: 在分析页面查看统计图表和协议分析
3. **导出数据**: 选择时间范围和格式导出通信数据
4. **生成报告**: 生成详细的通信分析报告

## 🔧 核心模块

### 串口管理 (SerialPortManager)
- 串口扫描和检测
- 连接和断开管理
- 参数配置和优化

### 数据处理 (DataProcessor)
- 实时数据接收和处理
- 多格式数据展示
- 数据过滤和搜索

### 可视化 (ChartRenderer)
- 实时流量图表
- 统计分析图表
- 协议层次展示

### 用户界面 (UIManager)
- 响应式布局管理
- 动画效果控制
- 通知和提示系统

## 📊 性能指标

- **数据处理能力**: 支持高达921600 bps的波特率
- **实时监控**: 毫秒级数据更新响应
- **内存管理**: 智能缓冲区管理，防止内存泄漏
- **兼容性**: 支持主流串口设备和协议

## 🔒 安全特性

- **本地处理**: 所有数据处理在浏览器本地完成
- **隐私保护**: 不收集或上传用户数据
- **权限管理**: 遵循浏览器串口访问权限规范
- **安全传输**: 支持安全的数据导出和分享
